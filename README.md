# LanternMing 上元明朝
一款涵盖目前 Unicode （Unicode 15.1）收录的所有汉字的旧字形风格中文字体，基于醍醐书体与花园明朝制作。

## 说明
* 完整涵盖了 Unicode 15.1 中的所有汉字。
* 中文优先使用醍醐明朝的旧字形字图。
* 继承了花园明朝丰富的非中文字符。
* 加入了少量 OpenType 功能。
* 与醍醐书体/花园明朝相同，本字体使用了 2 个字体文件。
