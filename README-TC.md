# LanternMing 上元明朝
一款涵蓋目前 Unicode （Unicode 15.1）收錄的所有漢字的舊字形風格中文字型，基於醍醐書體與花園明朝製作。

## 說明
* 完整涵蓋了 Unicode 15.1 中的所有漢字。
* 中文優先使用醍醐明朝的舊字形字圖。
* 繼承了花園明朝豐富的非中文字元。
* 加入了少量 OpenType 功能。
* 與醍醐書體/花園明朝相同，本字型使用了 2 個字型檔。
